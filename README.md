### ðŸš€ How to Compile the Codex

If you edit any file in `manuscripts/`, you must re-compile the master document to see changes reflected there.

1.  Open a terminal in this root directory.
2.  Run the compilation script:
    ```bash
    python compile_codex.py
    ```
3.  The script will:
    *   Read all files from `manuscripts/` in the correct order.
    *   Generate a dynamic Table of Contents.
    *   Output the result to `Ohu_Legacy_Codex_Compiled.md`.
