# Google Ads Copy Template

## Purpose

This template provides frameworks for writing effective Google Search and Display ad copy that drives clicks and conversions.

## Instructions

1. **Keyword Research**: Base copy on target keywords
2. **Match Intent**: Align copy with search intent
3. **Highlight Benefits**: Focus on what user gets, not features
4. **Include CTA**: Clear call-to-action in every ad
5. **Test Variations**: Create multiple versions for A/B testing

## Template Structure

### Google Search Ad (Standard)

**Headline 1** (30 characters):
> [BENEFIT OR KEYWORD]

**Headline 2** (30 characters):
> [DIFFERENT BENEFIT OR VALUE PROPOSITION]

**Headline 3** (30 characters):
> [CTA OR ADDITIONAL BENEFIT]

**Description 1** (90 characters):
> [EXPAND ON BENEFIT] [SUPPORTING DETAIL] [CTA]

**Description 2** (90 characters):
> [ADDITIONAL VALUE] [PROOF POINT] [URGENCY OR OFFER]

**Final URL**: [LANDING PAGE URL]

**Display Path** (optional):
- Path 1: [KEYWORD]
- Path 2: [BENEFIT]

## Best Practices

### Headlines
- **Use Keywords**: Include target keywords naturally
- **Focus on Benefits**: What user gets, not what you do
- **Create Urgency**: Limited time, limited availability
- **Test Numbers**: "Save 50%" vs "Save $500"

### Descriptions
- **Expand on Headlines**: Provide more detail
- **Include Proof**: Numbers, testimonials, guarantees
- **Clear CTA**: Tell user exactly what to do
- **Use Extensions**: Utilize ad extensions for more space

### Extensions
- **Sitelink Extensions**: Link to key pages
- **Callout Extensions**: Additional benefits
- **Structured Snippets**: Features or services
- **Call Extension**: Phone number for mobile

---

## A/B Testing Framework

Test one element at a time:

**Test 1: Headlines**
- Version A: Benefit-focused
- Version B: Problem-focused

**Test 2: CTAs**
- Version A: "Get Started"
- Version B: "Free Consultation"

**Test 3: Offers**
- Version A: Percentage discount
- Version B: Dollar amount discount
