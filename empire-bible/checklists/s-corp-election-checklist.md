## 2. Filing Requirements
- [ ] **Form 2553**: Download current version from IRS.gov.
- [ ] **Deadline**: Must be filed within 2 months and 15 days of the beginning of the tax year (usually March 15).
- [ ] **Signatures**: All shareholders must sign.

## 3. Payroll Setup (Required for S-Corps)
- [ ] **Reasonable Salary**: Determine reasonable compensation for officers.
- [ ] **Payroll Provider**: Set up Gusto/ADP.
- [ ] **State Registration**: Register for state unemployment/withholding taxes.

## 4. Maintenance

- [ ] **K-1s**: Issue K-1s to shareholders.
