# Signing Agent Checklist (The Perfect Signing)

## Purpose
To ensure zero errors in loan signing packages.

## Pre-Signing
- [ ] **Docs Received**: Downloaded and reviewed for special instructions.
- [ ] **Printed**: Dual-tray printing (Legal/Letter correct).
- [ ] **QC Check**: Verified names, dates, and property address on first page.
- [ ] **Flagged**: Sticky notes placed on all signature/initial lines.
- [ ] **Confirm**: Called borrower to confirm time/location and ID requirements.

## At the Table
- [ ] **ID Verified**: Checked unexpired Driver's License/Passport.
- [ ] **Journal**: Borrower signed journal.
- [ ] **Oath**: Administered oath for affidavits.
- [ ] **The Critical 5**:
    - [ ] **Note**: Signed?
    - [ ] **Deed of Trust/Mortgage**: Signed & Notarized?
    - [ ] **Closing Disclosure (CD)**: Signed?
    - [ ] **1003 Application**: Signed?
    - [ ] **W-9**: Signed?
- [ ] **Right to Cancel**: Dates correct? (3 business days). Borrower signed?

## Post-Signing
- [ ] **Scan-Backs**: Scanned critical docs and uploaded (if required).
- [ ] **Drop-Off**: Dropped at FedEx/UPS before cutoff.
- [ ] **Receipt**: Saved drop-off receipt.
