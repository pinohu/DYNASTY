# Capital Allocation Decision Checklist

## Purpose
This checklist ensures systematic evaluation of all capital allocation decisions for PNR Holdings LLC.

## Pre-Decision

- [ ] Identify capital allocation opportunity
- [ ] Gather initial information
- [ ] Assess preliminary fit
- [ ] Determine if worth pursuing

## Decision Criteria Evaluation

### Return on Investment

- [ ] Calculate expected return
- [ ] Assess risk-adjusted return
- [ ] Evaluate time to return
- [ ] Assess return sustainability
- [ ] Score: [1-10]

### Strategic Alignment

- [ ] Assess alignment with strategy
- [ ] Evaluate support for empire goals
- [ ] Check fit with entity structure
- [ ] Assess position enhancement
- [ ] Score: [1-10]

### Risk Assessment

- [ ] Identify risk level
- [ ] Assess risk mitigation
- [ ] Evaluate risk tolerance
- [ ] Review risk management
- [ ] Score: [1-10]

### Resource Requirements

- [ ] Calculate capital required
- [ ] Assess time commitment
- [ ] Evaluate management resources
- [ ] Review operational impact
- [ ] Score: [1-10]

### Opportunity Cost

- [ ] Identify alternative uses
- [ ] Assess foregone opportunities
- [ ] Evaluate best use of capital
- [ ] Consider optimization
- [ ] Score: [1-10]

## Approval Process

### Level 1: Manager Approval (Up to $10,000)

- [ ] Manager reviews proposal
- [ ] Manager evaluates criteria
- [ ] Manager makes decision
- [ ] Manager documents decision

### Level 2: Member Approval ($10,000-$50,000)

- [ ] Manager prepares proposal
- [ ] Members review proposal
- [ ] Members evaluate criteria
- [ ] Members make decision
- [ ] Members document decision

### Level 3: Unanimous Approval ($50,000+)

- [ ] Manager prepares comprehensive proposal
- [ ] All members review proposal
- [ ] All members evaluate criteria
- [ ] Unanimous approval required
- [ ] All members document decision
