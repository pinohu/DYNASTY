# Chapter 20 Implementation Resources

This directory contains the implementation resources for **Chapter 20: Automation Architecture** of the The Legacy Codex.

## Contents

- **`chapter-20-implementation-guide.md`**: The master step-by-step guide for building the automation infrastructure.
- **`activepieces-docker-compose.yml`**: Docker Compose configuration for self-hosting Activepieces.
- **`activepieces-nginx.conf`**: Nginx server block configuration for reverse proxy and SSL.
- **`custom-functions.js`**: JavaScript code for custom automation functions (Lead Scoring, Validation, Duplicate Check).
- **`workflow-definitions.md`**: Detailed logic and steps for KonnectzIT and Make.com workflows.
- **`integration-map.md`**: Documentation of system interconnections and data flow diagrams.

## How to Use

1.  Open `chapter-20-implementation-guide.md`.
2.  Follow Phase 1 to set up KonnectzIT workflows using `workflow-definitions.md`.
3.  Follow Phase 2 to deploy Activepieces using the provided Docker and Nginx files.
4.  Use `custom-functions.js` when implementing code steps in Activepieces or Make.com.
5.  Refer to `integration-map.md` to understand how data flows between systems.
