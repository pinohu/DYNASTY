# Cloudshot Research Report

## 1. Overview

**Name:** Cloudshot  
**Creator/Company:** Cloudshot (Founded by Sudeep Khire)  
**Website:** https://cloudshot.io  
**AppSumo Deal Status:** SOLD OUT  
**Core Purpose:** Minimalistic CloudOps platform for cost optimization, providing full-stack visibility across AWS, Azure, and GCP in a single visual platform, designed to replace scattered cloud management tools with unified real-time monitoring and control

**Key Value Proposition:** AI-powered cloud infrastructure management platform that visualizes, monitors, controls, and optimizes multi-cloud environments through intuitive drag-and-drop interfaces, automated Terraform script generation, and real-time cost optimization.

## 2. Primary Features & Capabilities

### Multi-Cloud Visualization & Management
- **Unified Dashboard:** Single visual platform for AWS, Azure, GCP, and Digital Ocean
- **Real-Time Topology View:** Dynamic network diagrams that update automatically
- **Drag-and-Drop Interface:** Intuitive draw.io-based interface for infrastructure management
- **Interactive Maps:** Visual heatmaps for fast anomaly detection
- **Cross-Cloud Search:** Filter by tags, owner, cost center, or service across all clouds
- **Live Status Updates:** Real-time color-coded status (red/amber/green) for all resources
- **Drill-Down Capabilities:** Deep dive into specific resources and configurations
- **Visual Security Overlays:** Security risk visualization on live infrastructure maps

### Automated Infrastructure Deployment
- **Terraform Script Generation:** Automatically generate Terraform scripts from visual diagrams
- **Infrastructure as Code:** Convert visual designs to deployable code
- **Import Script Generation:** Automatic "terraform import" command generation
- **Forward Engineering:** Create infrastructure from diagrams
- **Reverse Engineering:** Generate diagrams from existing infrastructure
- **Deployment Automation:** Streamlined deployment processes with minimal manual intervention
- **Configuration Management:** Manage infrastructure configurations visually
- **Version Control Integration:** Connect with existing Terraform and audit systems

### Cost Optimization & Financial Management
- **Real-Time Cost Monitoring:** Live tracking of cloud spending across all providers
- **Zombie Resource Detection:** Identify idle VMs, unattached storage, and orphaned instances
- **Cost Breakdown Analysis:** Visual spend breakdown by service, region, or team
- **Budget Alerts:** Forecast variance and budget monitoring
- **Waste Elimination:** Proactive identification of unnecessary expenses
- **Reserved Instance Recommendations:** Smart recommendations for reserved, spot, or light workloads
- **Cost Attribution:** Track costs by project, team, or business unit
- **Financial Reporting:** Comprehensive cost reporting and analytics

### Advanced Resource Management
- **Agentless Operations:** No need to install agents on servers
- **Bulk Operations:** Manage multiple resources simultaneously
- **Remote Command Execution:** Execute commands across multiple instances
- **System Information Gathering:** Collect system info without logging into individual machines
- **Third-Party Agent Management:** Install, stop, start, uninstall agents (e.g., Datadog)
- **Lifecycle Management:** Automated resource lifecycle policies
- **Performance Monitoring:** Real-time performance metrics and alerts
- **Health Checks:** Automated health monitoring across all resources

### Security & Compliance Management
- **IAM Drift Detection:** Identify IAM violations and policy drift
- **Security Risk Visualization:** Real-time security risk assessment
- **Compliance Monitoring:** Ensure adherence to security policies
- **Misconfiguration Detection:** Find misconfigured firewalls and security groups
- **Access Control:** Role-based access control and permissions management
- **Audit Trail:** Comprehensive logging and audit capabilities
- **Policy Enforcement:** Automated policy compliance checking
- **Vulnerability Assessment:** Continuous security vulnerability scanning

### Tagging & Governance Automation
- **Automated Tag Enforcement:** Standardize naming conventions and tagging
- **Tag Drift Detection:** Identify and fix missing or incorrect tags
- **Governance Policies:** Implement and enforce governance rules
- **Compliance Reporting:** Generate compliance reports automatically
- **Resource Classification:** Organize resources by business function
- **Cost Center Allocation:** Proper cost attribution through tagging
- **Lifecycle Policies:** Automated resource lifecycle management
- **Standardization Tools:** Ensure consistent resource management practices

### Team Collaboration & Management
- **Unlimited Team Members:** No restrictions on team size
- **Role-Based Access:** Granular permissions and access control
- **Multi-User Collaboration:** Real-time collaborative infrastructure management
- **Workspace Organization:** Separate environments and projects
- **Activity Tracking:** Monitor team activities and changes

- Excellent cost optimization and waste elimination capabilities

- Automated Terraform script generation from visual designs
- Strong real-time monitoring and security features
- Proven ROI with documented cost savings (e.g., $22,000/month savings)

**Significant Limitations:**
- **Major Restriction:** AppSumo deal limited to only 1 cloud account
- **Scalability Issues:** Not suitable for organizations with multiple cloud accounts
- **Limited Stacking:** Cannot purchase multiple codes for additional accounts
- **User Restrictions:** Unclear if limited to single user

**Best For:** Small to medium organizations with single cloud account setups who need comprehensive cloud infrastructure management, cost optimization, and visual infrastructure design capabilities.

**Not Suitable For:** Organizations with multiple cloud accounts, large enterprises with complex multi-account setups, or teams requiring extensive multi-account management.

**Key Consideration:** The single cloud account limitation significantly reduces the value proposition for most serious cloud users who typically operate multiple accounts for different environments (dev, staging, production) or business units.

**Value Rating:** 6.5/10 - Good platform with strong features but severely limited by single account restriction

The AppSumo lifetime deal offers good value for the right use case, but the single cloud account limitation makes it unsuitable for most organizations with mature cloud operations. The platform itself is solid with proven cost optimization capabilities, but the deal structure limits its applicability to smaller, simpler cloud setups.
