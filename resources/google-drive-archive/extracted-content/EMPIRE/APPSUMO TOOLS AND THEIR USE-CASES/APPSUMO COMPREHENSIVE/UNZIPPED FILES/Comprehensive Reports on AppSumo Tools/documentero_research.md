### Template Management:
- **Online Builder**: Create templates using built-in online editor
- **Upload Support**: Upload existing .docx/.xlsx templates from computer
- **Template Parsing**: Automatic analysis of fields, sections, and line items
- **Template Updates**: Automatic form/API updates when templates change

- **Shareable Forms**: Automatically generated web forms with customizable fields
