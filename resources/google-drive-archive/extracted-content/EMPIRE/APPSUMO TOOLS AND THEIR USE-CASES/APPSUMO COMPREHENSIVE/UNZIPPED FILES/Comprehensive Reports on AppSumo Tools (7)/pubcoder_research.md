## Product Overview

### Core Functionality
PubCoder is a desktop application (Mac/Windows) designed for creating interactive digital content that can be published across multiple platforms:

**Primary Capabilities:**
- Interactive ebook creation with animations, videos, and sounds
- Mobile app development for iOS and Android
- HTML5 content for web publishing
- EPUB3 and KF8 ebook export
- Multi-language and multi-screen adaptation

**Key Features:**
- Drag-and-drop editor with no coding required
- Touch gesture detection (clicks, swipes, shakes)
- Audio synchronization with text (read-aloud)
- Adobe InDesign importer
- Interactive widgets (games, quizzes, questionnaires)
- Animation panel with After Effects integration
- Multi-format export capabilities
- EPUB checker integration
- Preview on mobile app (PubReader)
