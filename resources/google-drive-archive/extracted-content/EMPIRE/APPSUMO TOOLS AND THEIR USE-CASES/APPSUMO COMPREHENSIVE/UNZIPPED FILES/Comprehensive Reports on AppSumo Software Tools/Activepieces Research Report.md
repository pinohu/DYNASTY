## Limitations

### Known Issues
1. **Learning Curve**: Requires understanding of automation concepts and logic
2. **Limited Mobile App**: No dedicated mobile app, web-only interface
3. **Task Limits**: Monthly task limits may be restrictive for high-volume users
4. **Sync Time Delays**: Scheduled triggers have minimum sync times (1-10 minutes)
5. **Community Size**: Smaller community compared to Zapier/Make

### Technical Limitations
- **JavaScript Required**: Advanced customizations require coding knowledge
- **Self-Hosting Complexity**: Self-hosted deployment requires technical expertise
- **API Rate Limits**: Subject to third-party API limitations and quotas
- **Browser Dependency**: Requires stable internet connection and modern browser

### Comparison Limitations
- **Fewer Integrations**: 330+ pieces vs 5000+ in Zapier
- **Less Mature**: Newer platform with fewer established workflows
- **Limited Enterprise Features**: Missing some advanced enterprise-grade features
- **Documentation Gaps**: Some integrations may lack comprehensive documentation
