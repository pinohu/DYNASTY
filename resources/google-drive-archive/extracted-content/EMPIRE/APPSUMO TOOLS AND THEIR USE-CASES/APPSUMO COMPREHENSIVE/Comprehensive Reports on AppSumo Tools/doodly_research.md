# Doodly Research

## Product Overview
**Doodly** is a whiteboard animation software that allows users to create professional doodle videos without any technical or design experience. It's the first and only doodle video software that can create whiteboard, blackboard, glassboard, and green screen animations using a simple drag-and-drop interface.
