## HOW TO CREATE AN AGENT

### Step 1: Select the Agent Prompt
Open `AGENT_PROMPTS_COMPLETE.md` and find the agent you want to create (e.g., "AGENT 1: VOLUME I WRITER")

### Step 2: Copy the Complete Prompt
Copy the entire prompt block (everything between the triple backticks ```markdown and ```)

### Step 3: Create the Agent
Use your agent creation system and paste the prompt as the agent's instructions/system prompt

### Step 4: Verify Resources
Ensure all source materials referenced in the prompt are available:
- Framework documents (`Empire Book Outline.txt`, `Trust Structure - Comprehensive Framework.md`, etc.)
- Extracted resources in `resources/google-drive-archive/extracted-content/EMPIRE/`

### Step 5: Activate the Agent
Start the agent and it will begin writing according to the prompt specifications

## TROUBLESHOOTING

### If an agent gets stuck:
1. Check if source materials are accessible
2. Verify file paths are correct
3. Ensure agent has proper permissions
4. Review quality checkpoints
5. Check for truncation issues

### If content quality is low:
1. Review writing standards section
2. Check source material references
3. Verify examples are being used
4. Ensure templates are being created
5. Review with Agent 12 (QA)

### If progress is slow:
1. Verify word count targets are reasonable
2. Check if agent is saving frequently
3. Ensure resources are accessible
4. Review workflow steps
5. Consider breaking into smaller tasks
