# Appendix A: Quick Reference Guides

## Introduction

This appendix provides quick reference guides for daily, weekly, monthly, quarterly, and annual tasks. These checklists ensure consistency and prevent critical tasks from being overlooked.

---

## Section A.1: Daily Checklist

### Executive
- [ ] Review North Star Dashboard (Cash, Sales, Ops).
- [ ] 15-min Standup with Leadership Team.
- [ ] Review "Top 3" Priorities for the day.
- [ ] Clear "Blockers" for the team.
- [ ] Zero Inbox (Triage, Delegate, Delete).

### Operations
- [ ] Check system uptime/status.
- [ ] Review customer support ticket queue.
- [ ] Verify daily backups.
- [ ] Process daily orders/transactions.

### Finance
- [ ] Review bank balances.
- [ ] Reconcile daily cash receipts.
- [ ] Approve pending payments (<$500).

---

## Section A.2: Weekly Task List

### Monday: Launch
- [ ] Team "War Table" Meeting (Review KPIs, Set Sprint Goals).
- [ ] Review Weekly Cash Flow Forecast.
- [ ] Marketing Performance Review (Ad Spend vs. ROAS).

### Friday: Retro
- [ ] Weekly Retrospective (What worked, what didn't).
- [ ] Celebrate "Wins" (Recognition).
- [ ] Clear desk/digital workspace for next week.

---

## Section A.3: Monthly Requirements

### Finance
- [ ] Close Books (Day 5).
- [ ] Review P&L vs. Budget.
- [ ] Process Payroll (if monthly).
- [ ] File Sales Tax (if monthly).

### Operations
- [ ] Vendor Performance Review.
- [ ] Inventory Audit.
- [ ] Facility Inspection.

### Governance
- [ ] Family Council Meeting (if monthly).
- [ ] Review Compliance Calendar.

---

## Section A.4: Quarterly Priorities

### Strategy
- [ ] Quarterly Business Review (QBR).
- [ ] Set OKRs for next quarter.
- [ ] Review 3-Year Roadmap.

### Finance
- [ ] File Quarterly Estimated Taxes.
- [ ] Board Meeting (Financial Presentation).
- [ ] Review Insurance Coverage.

### Talent
- [ ] Performance Reviews.
- [ ] Employee Satisfaction Survey (eNPS).

---

## Section A.5: Annual Calendar

### Q1 (Jan-Mar)
- [ ] Finalize Annual Budget.
- [ ] Kickoff Annual Strategy.
- [ ] Tax Prep (Send docs to CPA).

### Q2 (Apr-Jun)
- [ ] File Income Taxes (Apr 15).
- [ ] Annual Legal Compliance Audit.

### Q3 (Jul-Sep)
- [ ] Mid-Year Strategy Review.
- [ ] Begin Holiday Planning (Retail/E-com).

### Q4 (Oct-Dec)
- [ ] Annual Planning Offsite.
- [ ] Open Enrollment (Benefits).
- [ ] Year-End Tax Planning.
