# Client Onboarding Checklist

To ensure a smooth and professional start for every new client.

## Phase 1: Contract & Payment
- [ ] **Agreement Signed**: Received signed Service Agreement.
- [ ] **Deposit Paid**: Received initial payment.
- [ ] **CRM Entry**: Client profile created in SuiteDash.

## Phase 2: Information Gathering
- [ ] **Intake Form Sent**: Link to intake questionnaire emailed.
- [ ] **Documents Received**: All necessary files uploaded to portal.
- [ ] **Access Granted**: Logins for necessary accounts received.

## Phase 3: Kickoff
- [ ] **Welcome Email**: Sent "Welcome Aboard" email with expectations.
- [ ] **Kickoff Call**: Scheduled or completed (if applicable).
- [ ] **Team Assigned**: Task assigned to specific VA/Team member.

## Phase 4: Setup
- [ ] **Folder Created**: Client folder set up in Drive.
- [ ] **Slack/Communication**: Channel created (if applicable).
- [ ] **Project Plan**: Timeline and milestones mapped out.
