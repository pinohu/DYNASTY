# Loan Signing Agent Checklist

## Pre-Appointment
- [ ] **Confirmation**: Call borrower to confirm time, address, and ID requirements.
- [ ] **Documents**: Print 2 sets (one to sign, one for borrower).
- [ ] **Review**: Tag all signature, initial, and date lines.
- [ ] **Supplies**: Check bag (Stamp, Journal, Pens, Ink Pad, FedEx label).

## At the Table
- [ ] **ID Verification**: Check borrower's ID (Driver's License/Passport). Record in journal.
- [ ] **Journal Entry**: Have borrower sign the journal.
- [ ] **Oath**: Administer oath for affidavits.
- [ ] **The Signing**: Guide borrower through documents.
    - [ ] Deed of Trust / Mortgage
    - [ ] Note
    - [ ] Closing Disclosure (CD)
    - [ ] 1003 Loan Application
    - [ ] Tax Forms (4506-T)
- [ ] **Triple Check**: Review every page for missed signatures/dates BEFORE leaving.

## Post-Appointment
- [ ] **Scan-Backs**: Scan critical docs if required.
- [ ] **Drop-Off**: Drop package at FedEx/UPS. Get receipt.
- [ ] **Invoice**: Send invoice to Title Company/Signing Service.
- [ ] **Shred**: Shred any mistakes or extra copies securely.
