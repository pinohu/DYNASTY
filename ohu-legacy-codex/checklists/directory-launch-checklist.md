# Directory Launch Checklist

## Purpose
A step-by-step guide to launching a new niche directory on the Brilliant Directories platform.

## Phase 1: Research & Setup (Week 1)

- [ ] **Select Niche**: Confirm market demand and monetization potential.
- [ ] **Purchase Domain**: Buy a "keyword-rich" or "brandable" domain (e.g., `FindANotary.com`).
- [ ] **Set Up Brilliant Directories**:
    - [ ] Connect Domain.
    - [ ] Install SSL Certificate.
    - [ ] Configure Email Settings (SMTP).
    - [ ] Connect Payment Gateway (Stripe).
- [ ] **Design Configuration**:
    - [ ] Upload Logo.
    - [ ] Select Color Scheme.
    - [ ] Customize Homepage Hero Image.
    - [ ] Configure Main Menu.

## Phase 2: Content & Seeding (Week 2)

- [ ] **Define Categories**: Create the taxonomy (e.g., "Mobile Notary," "RON," "Loan Signing").
- [ ] **Import Locations**: Ensure all target cities/states are active.
- [ ] **Seed Listings**:
    - [ ] Scrape/Acquire list of 500-1,000 businesses.
    - [ ] Format CSV for import.
    - [ ] Import as "Free/Claimable" listings.
- [ ] **Generate SEO Content**:
    - [ ] Create 10 "Best [Service] in [City]" articles using AI.
    - [ ] Create "Join" page copy.
    - [ ] Create "About Us" page.

## Phase 3: Membership & Pricing (Week 3)

- [ ] **Define Tiers**:
    - [ ] Basic (Free/Claimable).
    - [ ] Premium (Monthly Fee).
    - [ ] Featured (High Ticket).
- [ ] **Set Pricing**: Configure pricing in BD dashboard.
- [ ] **Create Checkout Pages**: Test the signup flow.
- [ ] **Configure Lead Settings**:
    - [ ] Set lead price (if PPL).
    - [ ] Configure lead distribution rules.

## Phase 4: Launch & Promotion (Week 4)

- [ ] **Soft Launch**: Test all forms, links, and payments.
- [ ] **Claim Campaign**:
    - [ ] Email the 1,000 seeded businesses: "Your listing is live. Claim it now."
- [ ] **Social Media**:
    - [ ] Create FB/LinkedIn/Twitter pages.
    - [ ] Post launch announcement in industry groups.
- [ ] **Paid Ads (Optional)**:
    - [ ] Run small retargeting campaign to visitors.

## Phase 5: Post-Launch Optimization (Ongoing)

- [ ] **Monitor Traffic**: Google Analytics.
- [ ] **Monitor Signups**: Stripe Dashboard.
- [ ] **Solicit Feedback**: Email first 50 members.
- [ ] **Content Rhythm**: Post 1 new blog article per week.
