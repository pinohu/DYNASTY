# S-Corporation Election Checklist

## Purpose
This checklist ensures proper S-Corporation election for CXI LLC and ongoing compliance.

## Pre-Election

- [ ] Verify entity eligibility
  - [ ] Domestic corporation or LLC
  - [ ] Single class of stock/membership interests
  - [ ] 100 or fewer shareholders/members
  - [ ] Eligible shareholders/members only
- [ ] Verify shareholder/member eligibility
  - [ ] Individuals, certain trusts, or estates only
  - [ ] No partnerships, corporations, or non-resident aliens
- [ ] Review Operating Agreement
  - [ ] Ensure single class of interests
  - [ ] Verify transfer restrictions
- [ ] Obtain all shareholder/member consents

## Election Filing

- [ ] Prepare Form 2553
  - [ ] Complete all required information
  - [ ] Obtain all consents
  - [ ] Review for accuracy
- [ ] File Form 2553 with IRS
  - [ ] File within 75 days of entity formation OR
  - [ ] File within 75 days of start of tax year
  - [ ] Retain copy of filed form
  - [ ] Confirm receipt

## Post-Election

- [ ] Receive election confirmation from IRS
- [ ] Update entity records
- [ ] Notify shareholders/members
- [ ] Set up S-Corp compliance system

## Ongoing Compliance

### Annual Requirements

- [ ] File Form 1120-S annually
  - [ ] Due by March 15 (or extension)
  - [ ] Provide K-1s to shareholders/members
- [ ] Maintain eligible shareholders/members only
- [ ] Maintain single class of stock/interests
- [ ] Maintain active business operations
- [ ] Monitor compliance monthly

### Quarterly Requirements

- [ ] Review shareholder/member eligibility
- [ ] Review stock/interest structure
- [ ] Assess business activity
- [ ] Monitor compliance

### Monthly Requirements

- [ ] Monitor shareholder/member eligibility
- [ ] Review compliance status
- [ ] Update records as needed

## Reasonable Compensation

- [ ] Conduct compensation analysis annually
  - [ ] Market comparison
  - [ ] Cost approach
  - [ ] Income approach
- [ ] Document compensation decision
- [ ] Set reasonable salary ($60,000-$80,000)
- [ ] Pay salary consistently
- [ ] Document all compensation

## Distribution Planning

- [ ] Plan distributions quarterly
- [ ] Set distribution amounts
- [ ] Document distribution decisions
- [ ] Execute distributions
- [ ] Document distributions
- [ ] Report on K-1s

## Documentation

- [ ] Maintain Form 2553 copy
- [ ] Maintain election confirmation
- [ ] Document compensation analysis
- [ ] Maintain distribution records
- [ ] Retain all tax returns
- [ ] Maintain compliance records

## Notes

- S-election must be filed within 75 days of formation or tax year start
- All shareholders/members must consent
- Must maintain eligibility continuously
- Reasonable compensation is critical for compliance
- Distributions are tax-efficient (no FICA)
