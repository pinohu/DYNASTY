# Wyoming LLC Formation Checklist

## Purpose
This checklist ensures complete and proper formation of Wyoming LLCs for the Legacy Codex.

## Pre-Formation

- [ ] Choose LLC name
- [ ] Check name availability (Wyoming Secretary of State)
- [ ] Reserve name if needed (optional)
- [ ] Select registered agent
- [ ] Determine management structure (Member-Managed or Manager-Managed)
- [ ] Determine initial members and ownership percentages
- [ ] Plan capital contributions

## Formation Documents

- [ ] Prepare Articles of Organization
  - [ ] LLC name
  - [ ] Registered agent name and address
  - [ ] Principal office address
  - [ ] Management structure
  - [ ] Organizer information
  - [ ] Duration (perpetual or specific term)
- [ ] Review Articles for accuracy
- [ ] Sign Articles

## Filing

- [ ] File Articles of Organization with Wyoming Secretary of State
  - [ ] Online filing (recommended) OR
  - [ ] Mail filing OR
  - [ ] In-person filing
- [ ] Pay filing fee ($60)
- [ ] Receive confirmation
- [ ] Download filed documents

## Post-Formation

- [ ] Obtain EIN (Form SS-4)
  - [ ] Apply online (recommended)
  - [ ] Receive EIN immediately
- [ ] Create Operating Agreement
  - [ ] Include all required provisions
  - [ ] Customize for entity needs
  - [ ] Execute Operating Agreement
- [ ] Open business bank account
  - [ ] EIN required
  - [ ] Articles of Organization
  - [ ] Operating Agreement
  - [ ] Member resolutions (if needed)
- [ ] File Initial Report (if required)
- [ ] Obtain business licenses (if needed)
- [ ] Register for taxes (if required)
- [ ] Set up accounting system

## Series LLC (If Applicable)

- [ ] Include Series LLC provisions in Articles
- [ ] Include Series LLC provisions in Operating Agreement
- [ ] Create Series structure
- [ ] Designate assets to each Series
- [ ] Maintain separate books and records
- [ ] Maintain separate bank accounts

## Compliance Setup

- [ ] Set up compliance calendar
- [ ] Schedule annual report filing
- [ ] Set up registered agent relationship
- [ ] Establish record-keeping system
- [ ] Set up tax compliance system

## Documentation

- [ ] Organize formation documents
- [ ] File documents securely
- [ ] Create entity file
- [ ] Maintain complete records
- [ ] Update entity records as needed

## Notes

- Formation typically takes 1-3 business days for online filing
- EIN can be obtained immediately online
- Operating Agreement should be created within 90 days of formation
- All documents should be retained permanently
