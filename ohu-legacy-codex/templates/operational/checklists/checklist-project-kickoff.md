# Project Kickoff Checklist

## Purpose

This checklist ensures that every new project starts with clarity, alignment, and all necessary resources. A strong kickoff prevents scope creep and miscommunication later.

## Instructions

1.  **Timing**: Complete this checklist *after* the contract is signed but *before* work begins.
2.  **Attendees**: Involve the internal team and key client stakeholders.

## Checklist

### 1. Internal Preparation
- [ ] **Contract Review**: Review SOW, deliverables, and timeline.
- [ ] **Team Assignment**: Assign Project Manager and delivery team.
- [ ] **Resource Setup**: Create project folder (Drive/SharePoint) and Slack/Teams channel.
- [ ] **Tool Setup**: Set up project in Asana/ClickUp/Jira.
- [ ] **Budget Review**: Confirm budget and hours allocation.

### 2. Client Onboarding
- [ ] **Welcome Email**: Send welcome email with kickoff meeting invite.
- [ ] **Information Request**: Send questionnaire or request for access (logins, assets).
- [ ] **Invoicing**: Send deposit invoice (if applicable).

### 3. Kickoff Meeting Agenda
- [ ] **Introductions**: Meet the team.
- [ ] **Goals**: Reiterate project objectives and success metrics.
- [ ] **Scope**: Review deliverables and what is *out* of scope.
- [ ] **Timeline**: Review key milestones and deadlines.
- [ ] **Communication**: Agree on update frequency and channels.
- [ ] **Roles**: Clarify who approves work and who provides feedback.
- [ ] **Next Steps**: Immediate action items for both sides.

### 4. Post-Kickoff
- [ ] **Meeting Notes**: Send summary of kickoff meeting to client.
- [ ] **Timeline Finalization**: Adjust timeline based on meeting feedback.
- [ ] **Status**: Move project status to "Active".
