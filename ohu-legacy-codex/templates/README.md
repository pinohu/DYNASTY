## Template Standards

All templates follow these standards:

1. **Functionality**: All templates are fillable and usable
2. **Completeness**: No placeholders - all fields clearly defined
3. **Instructions**: Each template includes usage instructions
4. **Examples**: Example filled versions provided where helpful
5. **Version Control**: All templates are versioned
6. **Format**: Markdown with clear structure, tables for forms

## How to Use Templates

1. **Locate Template**: Use the Template Index to find the template you need
2. **Review Instructions**: Read the template's purpose and instructions section
3. **Customize**: Replace all [FIELD NAME] placeholders with actual information
4. **Review**: Have qualified professionals review legal/financial templates
5. **Execute**: Use the template as needed
6. **Document**: Keep executed templates in appropriate records

## Legal Disclaimer

**IMPORTANT**: All legal templates must be reviewed and customized by qualified legal counsel before execution. These templates serve as starting points and must be adapted to specific circumstances, state laws, and current regulations.
