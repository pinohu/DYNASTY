# 12-Month Financial Forecast Template

## Purpose

This template provides a comprehensive 12-month financial projection, integrating the Income Statement (P&L), Balance Sheet, and Cash Flow. It is the "master model" for financial planning.

## Instructions

1.  **Assumptions**: Start by defining key drivers (growth rate, churn, pricing).
2.  **P&L**: Project revenue and expenses.
3.  **Cash Flow**: Adjust P&L for timing differences (AR/AP) and CapEx.
4.  **Balance Sheet**: Calculate resulting asset/liability balances.
