# Quarterly Tax Projection Template

## Purpose

This template helps estimate quarterly estimated tax payments. It prevents underpayment penalties and cash flow surprises at tax time.

**IMPORTANT**: This is an estimate. Consult a CPA for actual filings.

## Instructions

1.  **Actual YTD**: Enter actual profit for completed quarters.
2.  **Projected**: Estimate profit for remaining quarters.
3.  **Tax Rate**: Apply estimated effective tax rate (Federal + State).
4.  **Payments**: Record payments made to track balance due.
